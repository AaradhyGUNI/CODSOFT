# Handwritten Text Generation

This project generates text character-by-character using a Long Short-Term Memory (LSTM) neural network. The model is trained on a text dataset and learns patterns in the text to generate new sequences based on user-provided input.

## Features

- Text preprocessing and cleaning
- Character-level text generation
- LSTM-based deep learning model
- User-defined text prompts
- Trained model saving and reuse
- Text prediction using TensorFlow/Keras

## Technologies Used

- Python
- NumPy
- TensorFlow
- Keras
- LSTM Neural Network
- Google Colab
- Jupyter Notebook

## Dataset

The model was trained on a text dataset containing writing prompts and human-written stories. The data was preprocessed, encoded into numerical format, and used to train an LSTM model for sequence prediction.

## Workflow

1. Load and preprocess text data
2. Convert characters into numerical values
3. Create input-output sequences
4. Build and train the LSTM model
5. Save the trained model
6. Generate text from custom user prompts

## Output Example

Input Prompt:

King

Generated Text:

king and the kingdom was surrounded by mysterious creatures and hidden secrets...
